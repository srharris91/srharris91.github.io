Information

This code will compute the Parabolic Stability Equation (PSE) for various flows. This will read in the Orr-Sommerfeld solution from a python script, and will march it forward in \(x\).

All of the code is contained in the namespace PSE.

The PSE equation is defined here

\[ A \hat{q}_{n,m} + B \frac{\partial \hat{q}_{n,m}}{\partial y} + C \frac{\partial^2 \hat{q}_{n,m}}{\partial y^2} + D \frac{\partial \hat{q}_{n,m}}{\partial x} = F \]
